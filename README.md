# 🌐 Portofolio — Tubagus Khairul Zaman

Portfolio pribadi yang dibangun dengan HTML, CSS, dan JavaScript murni (no framework).

## ✨ Fitur

- 🖥️ Terminal boot animation dengan typewriter effect
- 📊 Scroll progress bar
- 🔄 Skeleton loading screen
- 📦 Proyek diambil otomatis dari GitHub API
- 🌙 Dark mode (GitHub-inspired dark theme)
- 📱 Responsive design
- ♿ Accessible (keyboard navigation, aria-live, reduced-motion support)

## 🚀 Cara Pakai

1. Clone repo ini
2. Buka `index.html` di browser
3. Edit objek `CONFIG` di bagian bawah file untuk menyesuaikan data

```js
const CONFIG = {
  name: "Nama Kamu",
  role: "Role / Jabatan",
  bio: "Bio singkat kamu",
  githubUsername: "username",
  email: "email@example.com",
  linkedinUrl: "https://linkedin.com/in/...",
  skillCategories: [...]
};
```

## 🛠️ Tech Stack

- HTML5 Semantic
- CSS3 (custom properties, grid, animations)
- Vanilla JavaScript (fetch API, DOM manipulation)
- Google Fonts (JetBrains Mono + IBM Plex Sans)
- GitHub REST API
